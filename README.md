# Penguin Tests
POC sandbox to test out pytest functionality with a UI.

# VSC / Cursor
## Running tests

| Commands          | Description                                                                             |
| ----------------- | --------------------------------------------------------------------------------------- |
| `cmd + shift + d` | Open the debugger panel. There will be a select box near the top of the explorer panel. |
| `f5`              | Execute selected configuration while pausing for all active breakpoints                 |
| `ctrl + f5`       | Executed selected configuration ignoring all breakpoints                                |
